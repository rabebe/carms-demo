# CaRMS Demo - Junior Data Scientist

This project demonstrates a small data platform using the CaRMS public data.

Stack used:
- PostgreSQL
- SQLAlchemy / SQLModel
- FastAPI
- Dagster
- LangChain
- Docker (containerized)
- Optional AWS Free Tier deployment

## Setup

1. Ensure Docker is installed.
2. Run:
   ```bash
   docker-compose up --build
   ```
